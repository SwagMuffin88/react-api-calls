# Filmide info rakendus TMDB API-ga
React abil ehitatud rakendus, mis kasutab TMDB API-t filmide info pärimiseks ja dünaamiliselt kuvamiseks.
Rakenduses saab lehitseda populaarsemaid filme, kasutada otsingufunktsiooni ja filtreerida
filme žanrite järgi. Kõik andmed filmide kohta tulevad TMDB andmebaasist.

Rohkem TMDB API kohta: [link](https://developer.themoviedb.org/docs/getting-started).

Rakenduse jaoks kasutatud API kood pärineb järgnevast YT õpetusvideost:
[link](https://www.youtube.com/watch?v=PRUTl0ihzHg).

## Käivitamisjuhend
Veendu, et su arvutis on olemas Node.js ja npm. Alustamiseks klooni repo:
```
git clone https://github.com/SwagMuffin88/react-api-calls.git
cd react-api-calls
```

Seejärel projekti juurkaustas:
```
npm install
npm run dev
```

Peale käivitamist on rakendus kättesaadav aadressil http://localhost:5173/.
